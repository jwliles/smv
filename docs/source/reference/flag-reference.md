# Flag Reference

Complete reference for all SMV flags and options.

## Core Flags

| Flag | Description |
|------|-------------|
| `-p` | Preview mode |
| `-F` | Force mode |
| `-r` | Recursive |
| `-e` | Everything (include directories) |

## Additional Flags

| Flag | Description |
|------|-------------|
| `-i` | Case-insensitive |
| `-v` | Verbose |
| `-a` | Include hidden files |
| `-n` | No-clobber |
