# Transformations

Detailed guide to SMV's file transformation capabilities.

## Case Transformations

### snake_case
```bash
smv snake . -p
```

### kebab-case
```bash  
smv kebab . -p
```

### PascalCase
```bash
smv pascal . -p
```

### camelCase
```bash
smv camel . -p
```

### Title Case
```bash
smv title . -p
```

### lowercase
```bash
smv lower . -p
```

### UPPERCASE
```bash
smv upper . -p
```

## String Operations

### CHANGE Command
```bash
smv CHANGE "old" INTO "new" . -p
```
