# File Operations

Guide to SMV's file operation commands.

## Move Operations
```bash
smv mv source.txt dest.txt
```

## Copy Operations  
```bash
smv cp source.txt backup.txt
```

## Delete Operations
```bash
smv rm . EXT:log -p
```

## Creation Operations
```bash
smv -cd newdir
smv -cf newfile.txt
```
